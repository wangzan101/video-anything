# 黄金样例:提炼要点(key-points)

这份样例给「提炼要点」playbook(见 `SKILL.md` 第 4 步)一个具体的质量锚点:
输入长什么样、期望产出长什么样。下面两份「输入」是 `scripts/fetch.sh` +
`scripts/transcribe.py` 真实会产出的文件格式(节选),「期望产出」是
agent 应该写入 `summary.md` 的内容。

## 输入 1:`info.json`(节选)

```json
{
  "id": "7123456789012345678",
  "title": "自由职业三年,我总结的4个反拖延技巧",
  "uploader": "阿凯的效率日记",
  "duration": 250,
  "description": "别再靠意志力硬扛了,这四个方法我自己用了快两年,亲测有效。",
  "webpage_url": "https://www.douyin.com/video/7123456789012345678",
  "tags": ["效率", "自我提升", "拖延症"]
}
```

## 期望产出:`summary.md`

```markdown
# 自由职业三年,我总结的4个反拖延技巧

## 主旨
分享作者亲测两年、真正治好重度拖延症的四个反拖延技巧,核心思路是"降低开始的门槛",而不是硬靠意志力。

## 要点
- 拖延的根源常常不是意志力不够,而是"开始"这个动作本身太难 [00:22]
- 两分钟启动法:任何任务先只做两分钟,允许自己两分钟后停下,通常很快就会进入状态不想停 [00:26]
- 把手机放到另一个房间而不是只调静音,物理隔离比心理克制有效得多 [01:02]
- 每天早上写下最重要的三件事(不多不少),完成率能明显提升 [01:35]
- 用"及格线交付"对抗完美主义型拖延:先交出六十分版本,再逐步改进 [02:40]
- 如果只能记住一个方法,优先试「两分钟启动法」,因为它成本最低、最容易今天就上手 [03:45]

## 一句话金句
> "拖延的反义词不是自律,是开始。" [04:04]
```

## 为什么这是"好"的提炼(不是逐句翻译)

- **主旨是一句话概括,不是罗列**:没有把四个技巧都塞进主旨,而是提炼出它们共同的底层逻辑。
- **每条要点是"信息点"而不是"原句复述"**:比如 `[00:34]`+`[00:45]` 两句原文被合并成一条要点(两分钟启动法的做法+效果),而不是照抄逐句译文。
- **出处取的是该信息点在原文里最早出现的时间戳**,方便用户回看原片核对。
- **金句是原话直引**,不是转述,且单独放在末尾而不是混进要点列表。
- **要点数量落在 3–7 条区间**(本例 6 条),覆盖了视频的全部实质内容,没有为了凑数硬拆,也没有漏掉技巧四。
